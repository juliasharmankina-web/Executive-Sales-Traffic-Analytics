# Exploratory-data-analysis-for-online-store
Leverage SQL, Python, and Tableau to build a comprehensive analytics suite that transforms raw e-commerce data into actionable insights on sales dynamics, customer segmentation, and statistical performance drivers.

## Business Task
Perform a comprehensive diagnostic of the online store's performance by transforming raw database records into a strategic analytical report that identifies the primary drivers of revenue and user engagement.

## Key Insights
- Sessions and revenue are nearly perfectly correlated.
- Global markets move in sync.
- Marketing channels perform in tandem.
- User registration status impacts purchasing habits.
- Traffic quality varies significantly by source.
- Statistical analysis helps identify important patterns in sales data.

## Tools
SQL, Python (Pandas, NumPy), Data Visualization (Matplotlib, Seaborn, Tableau), Exploratory Data Analysis (EDA), Statistical Analysis
