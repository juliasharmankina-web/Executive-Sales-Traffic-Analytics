# Executive-Sales-Traffic-Analytics
Leverage SQL, Python, and Tableau to build a comprehensive analytics suite that transforms raw e-commerce data into actionable insights on sales dynamics, customer segmentation, and statistical performance drivers.

## Business Task
This project focuses on the comprehensive analysis of sales data, encompassing data cleaning, exploratory data analysis, and advanced statistical evaluations. The core objectives include understanding sales trends across various dimensions, comparing sales behavior between different user segments, and identifying significant correlations.

## Key Insights
- A deeper understanding of key sales drivers.
- The identification of geographical and channel-specific performance patterns.
- Informed decision-making for marketing strategies and user engagement initiatives.


## Tools
SQL, Python (Pandas, NumPy), Data Visualization (Matplotlib, Seaborn, Tableau), Exploratory Data Analysis (EDA), Statistical Analysis
